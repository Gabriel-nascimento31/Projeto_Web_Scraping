# Projeto_Web_Scraping

Esse projeto foi desenvolvido na disciplina Linguagem da Programação 2, com o tema Web Scraping.
Esse é um script Python que coleta as principais notícias do portal do G1:
Ele coleta o Título, o Link e o Resumo que é uma pequena descrição da notícia se estiver disponível

Foram utilizadas as seguintes bibliotecas nesse projeto:

Requests: Utilizada para fazer a requisição da URL da pagína na WEB 
Beautifulsoulp: Utilizada para analisar o código fonte HTML da pagína, coletar os dados especificos da pagína, que no caso desse projeto foram os títulos, os links e as descrições das notícias.

Para rodar o projeto instale as bibliotecas que estão no arquivo requirements.txt e execute o arquivo scraping.py
